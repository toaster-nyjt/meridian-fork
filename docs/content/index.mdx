# Meridian: A Design Framework for Malleable Overview-Detail Interfaces

Welcome to the Meridian documentation. Meridian is a powerful React-based framework for building malleable overview-detail interfaces (ODIs) using a declarative JSON specification.

## What is Meridian?

Meridian is a design framework that enables developers to create flexible, interactive data exploration interfaces. It provides:

- **Declarative JSON Specification**: Define your entire interface using a simple, type-safe JSON structure
- **Malleable Interfaces**: Built-in support for user customization of content, composition, and layout
- **Reusable Components**: Pre-built overview types (grid, list, map, table) and detail views
- **Data Binding**: Powerful attribute system with transformations, conditions, and nested structures
- **Type Safety**: Full TypeScript support with comprehensive type definitions

## Key Features

### Overview-Detail Pattern
Meridian implements the overview-detail pattern, allowing users to browse collections of data in an overview and dive into details when needed.

### Malleability
Users can customize three dimensions of the interface:
- **Content**: Toggle attributes on/off
- **Composition**: Switch between different views and layouts
- **Layout**: Arrange views in tabs, panels, or side-by-side

### Flexible Data Binding
The attribute system supports:
- Conditional rendering based on item properties
- Data transformations (map, filter, slice)
- Nested attributes and attribute groups
- Role-based attribute selection
- Custom attribute types

## Quick Example

```typescript
import { ODI } from 'meridian';

const myODI: ODI = {
  dataBinding: [{
    binding: {
      itemId: ".id",
      attributes: [
        { value: ".title", roles: ["title"] },
        { value: ".description", roles: ["description"] },
        { value: ".image", type: "image", roles: ["thumbnail"] }
      ]
    }
  }],
  overviews: [{
    type: "grid",
    detailViews: [{
      type: "basic",
      openFrom: ["title"]
    }]
  }]
};
```

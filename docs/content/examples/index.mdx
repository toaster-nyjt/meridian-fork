# Examples

This section provides complete, real-world examples of Meridian implementations. Each example includes full ODI specifications and demonstrates different features and patterns.

## Available Examples

### E-commerce Product Catalog
A complete product catalog with grid/list views, filters, and detailed product pages.

**Features:**
- Conditional pricing display
- Image galleries
- Product specifications
- Related products
- Ratings and reviews

[View Example →](/examples/e-commerce)

### Real Estate Listings
Property listings with map view, detailed property pages, and search filters.

**Features:**
- Map-based overview
- Price-based filtering
- Property specifications
- Image galleries
- Contact forms

[View Example →](/examples/real-estate)

### Social Media Feed
A social media style interface with posts, profiles, and interactions.

**Features:**
- Timeline view
- User profiles
- Interactive elements
- Nested comments
- Media attachments

[View Example →](/examples/social-media)

## Running the Examples

### Gallery App

```bash
cd examples/gallery
npm install
npm run dev
```

### Web Builder

```bash
cd examples/web-builder
npm install
npm run dev
```
